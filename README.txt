1.	Purpose of the application:
	The application allows you to save information about expenses in 3 categories: "food", "bills", "pleasures". Expenses are examined as part of monthly budgets.
	The application returns information about the values of these expenses within budgets in the indicated categories and informs you if a given budget within a given category has been exceeded.
	For the purposes of the application, I assumed that the budget limits for specific categories are as follows: "food - 2000", "bills - 1500", "pleasures - 1000".

2. 	User manual:
	After starting the application, information about any previously entered expenses will be read from the file, and then the information will be returned if budgets have been exceeded so far.
	Then enter information about the new expense. The required information is the expense name, expense value, expense date, expense category ("food", "bills", "pleasures").
	You can enter more than one expense.
	After entering all expenses, the program will present a summary of all expenses (previously saved and newly added) and return a summary showing expenses assigned to the correct year, month and category.
